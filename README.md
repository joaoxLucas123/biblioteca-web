# Sistema de Biblioteca

Projeto desenvolvido em React com TypeScript e Bootstrap.

## Objetivo
Criar um sistema de biblioteca que permita cadastrar, visualizar, alugar, devolver e excluir livros, com atualização dinâmica dos dados.

## Funcionalidades
- Cadastro de livros
- Listagem de livros
- Aluguel de livros
- Devolução de livros
- Exclusão de livros disponíveis
- Dashboard com contadores (total, disponíveis e alugados)

## Tecnologias utilizadas
- React
- TypeScript
- Bootstrap

## Estrutura do projeto
O sistema foi dividido em componentes para melhor organização:
- Header
- Dashboard
- FormLivro
- ListaLivros
- LivroCard
- Footer
